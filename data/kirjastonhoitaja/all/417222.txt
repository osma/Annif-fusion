Löytyykö muuta kirjallisuutta liittyen Java teknologiaan 
JMS ==> Java Messaging Service.
Kaipaamiasi kirjoja ei valitettavasti löytynyt Frank-monihaun http://monihaku.kirjastot.fi/frank/frankcgi.py/index avulla. 

Sen sijaan löytyi mm. seuraavanlaisia aihetta käsitteleviä kirjoja:
- Mark Wutka: Inside Java J2EE (2002) (saatavilla pääkaupunkiseudun yleisistä kirjastoista)
- John Hunt: Guide to J2EE: enterprise Java (2003) (saatavilla Haaga-Helia ammattikorkeakoulun kirjastosta)
- Markku Sulasma: JMS messaging in enterprise enviroment (2004) (saatavilla Haaga-Helia ammattikorkeakoulun kirjastosta, tämä on opinnäytetyö, kannattaa vilkaista työn lähdeluettelo, sieltä löytyy varmasti mielenkiintoisia lähteitä)
- Richard Monson-Haefel: Java message service (2001) (saatavilla Teknillisen korkeakoulun kirjastosta)
- Richard Monson-Haefel: Enterprise Javabeans: tehokäyttäjän opas (2002) (saatavilla Laurea ammattikorkeakoulun, Metropolia ammattikorkeakoulun ja Teknillisen korkeakoulun kirjastoista)

Tällaisten kirjojen löytämistä hankaloittaa hieman se, että teosten asiasanat ovat usein liian yleisellä tasolla. (Kun kirjaston tietokannassa tehdään hakuja, niin kirjat löytyvät niiden sisältöä kuvaavien asiansanojen perusteella.) Esimerkiksi Inside Java J2EE (2002) -kirjan asiasanoiksi on Helmet-tietokannassa laitettu vain ohjelmointikielet, Java, ohjelmointi, WWW ja Internet. Asiasana J2EE puuttuu. Kun haet JMS-kirjoja kirjastojen tietokannoista, niin kannattaa kokeilla hakemista myös muilla rimpsuilla kuin "Java message service". Esimerkiksi J2EE-asiasanalla varustetut kirjat saattavat olla kannaltasi mielenkiintoisia. Kannattaa hakea myös uusimpia Java-oppaita.
